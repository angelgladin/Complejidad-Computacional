## Ejecución del programa

Se tiene un archivo `Makefile` para la ejecución y limpieza del programa.

Para ejecutar **Subset-sum**:
```bash
$ make run_subset
```

Para ejecutar **Bin Packing**:
```bash
$ make run_bin
```

Para limpiar:
```bash
$ make clean
```
